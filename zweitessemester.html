Mein Zweites Semester an der Technischen Hochschule Mittelhessen

Learn More about THM

In meinem zweiten Semester an der THM habe ich folgende Module am laufen

  • Internetbasierte Systeme

    In dieser Veranstaltung werden uns grundlegende Elemente der Web-Entwicklung beigebracht, wie z.B. git, github, html und CSS. Außerdem wird uns der Umgang mit Frameworks wie Twitter Bootstrap erläutert. Meine bisherige Leistung ist unter anderem in Badges dokumentiert und auf GitHub veröffentlicht.

    Mein Pull Request als Reviewer.

    Pull Request den ich erhalten habe.

  • Algorithmen und Datenstrukturen

    Hier wird uns erklärt wie man seinen Code am besten gestaltet, analysiert und die richtigen Datenstrukturen verwendet.

  • Lineare Algebra

    Vektoren, Matrizen, Determinanten, Eigenwerte.

  • Programmieren interatktiver Systeme

    Eigentlich ist "PiS" die Weiterführung von Objekt orientierte Programmierung aus dem ersten Semester. Wir benutzen hier das gelernte um Programme zu schreiben die eine grafische Oberfläche haben.

  • Datenverarbeitungsorientierte Betriebswirtschaftlehre

    Wie der Name schon sagt schauen wir uns hier die Arbeitsprozesse der Betriebswirtschaftlehre an und wie man sie durch Computer optimieren kann. Außerdem bekommen wir einen Einblick in die Struktur eines Unternehmens.

  • Recht für Informatiker

    Lehrinhalt sind die Grundlagen des Zivilrechts und Datenschutz und solche Sachen.
